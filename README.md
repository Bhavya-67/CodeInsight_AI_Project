# CodeInsight_AI_Project
🤖 CodeInsight AI – Intelligent Code Review Platform CodeInsight AI is a web-based AI-assisted code review platform that analyzes Python files for code quality, complexity, security vulnerabilities, and style issues. It helps developers improve code before deployment by providing automated insights and suggestions. 🌐 Live Application: https://gq52fhz8m2ajztcyyyt5id.streamlit.app/ 🚀 Features 🔎 Automated Code Analysis Upload a Python file and the system automatically analyzes it using multiple developer tools. 🧠 Code Health Score Displays an overall Code Health Score (0–100) based on: Style issues Code complexity Security vulnerabilities 🎨 Style Checking Uses Flake8 to detect: PEP8 violations formatting issues unused variables 📈 Complexity Analysis Uses Radon to measure: cyclomatic complexity maintainability of functions 🔐 Security Scanning Uses Bandit to detect: insecure coding patterns common security vulnerabilities ✨ Automatic Code Formatting Uses Black to provide a refactored version of the uploaded code. 📊 Interactive Dashboard Displays results using visual components including: health score gauge metrics dashboard before vs after code comparison 🖥️ Application Interface The platform includes: 📊 Dashboard overview 📂 Python file upload 🔍 Real-time code analysis 📈 Code complexity insights 🔐 Security vulnerability report ✨ Refactored code output
